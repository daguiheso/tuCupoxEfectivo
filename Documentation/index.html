Murun | Responsive App Landing Page Documentation

Murun is a responsive App landing template. It’s one page, clean and modern design. Murun is built with latest HTML5, CSS3 and Bootstrap 3+. The clean and unique design will amaze you. Whatever you are browsing from mobile tablet or desktop/laptop, the Murun will be fit all screen sizes. Murun has everything you need to launch your site!

HTML Structure

Sections

CSS File and Structure

CSS files inlude

  1. bootstrap.min.css
  2. animate.min.css
  3. themify-icons.css
  4. elegant-line-icons.css
  5. owl.carousel.min.css
  6. swipebox.min.css
  7. swiper.min.css
  8. main.css
  9. responsive.css
  10. base.css

As like HTML file, CSS file is also well documented. You can find it in css > main.css file. It has table of content for every section. You don't need to modify the bootstrap, font awesome and magnific files. Just modify the style.css.

JavaScript

JavaScript files include

  1. base.js
  2. bootstrap.min.js
  3. jquery-1.12.4.min.js
  4. jquery.swipebox.min.js
  5. modernizer-2.8.3.min.js
  6. owl.carousel.min.js
  7. smooth-scroll.min.js
  8. swiper.min.js
  9. typed.min.js
  10. wow.js
  11. main.js

The main.js file is the main file for all JavaScript code. It's also well-documented. You can modify JavaScript file through reading the code comments.

Images and Assets
